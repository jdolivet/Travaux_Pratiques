Site WEB

Consignes

Le but de cet exercice est de créer un site WEB sur le personnage d'Alan Turing, figure centrale de l'informatique du 20e siècle.

Vous êtes libre d'utiliser les outils de votre choix (un éditeur de texte et un navigateur suffisent).

Voici le squelette d'un document HTML :

Le site sera constitué de 3 pages.

  • Une page index.html, qui sera, comme dans tout site web, la page d'accueil.
    En particulier, on y trouvera l'image Turing.jpg, des liens vers les autres pages de notre site et vers quelques autres sites sur Alan Turing.
  • Une page enigma.html qui donnera des informations sur la machine à coder Enigma et contiendra l'image Enigma.jpg.
  • Une page machine.html qui contiendra un petit texte sur les machines de Turing avec une image Machine.jpg.
Voici les étapes pour réaliser votre site :
  1. index.html.
    Créez la page index.html.
    Celle ci contient un court texte introductif et une image (centrée) d'Alan Turing (vous pouvez trouvez une image à cette adresse). Vous devrez utiliser l'URL absolue de l'image (l'image ne sera pas présente dans votre dossier).
    La page contient également des liens vers les autres pages de ce site (enigma.html et machine.html) ainsi que des liens vers le site de la cité des sciences et le site d'Arte.
  2. enigma.html.
    Créez la page enigma.html.
    Celle ci contient un court texte descriptif et une image (centrée) de la machine Enigma (vous pouvez trouvez une image à cette adresse). Vous devrez utiliser l'URL relative de l'image (l'image choisie devra donc être présente dans votre dossier).
    Vous devrez également construire un tableau (2 lignes, 2 colonnes) donnant un exemple de message codé à partir de la machine Enigma (ce site pourra vous être utile).
  3. machine.html.
    Créez la page machine.html.
    Celle ci contient un court texte descriptif et devra intégrer la vidéo suivante (le lien mp4 se trouve à cette adresse).

Il faut donc réunir vos pages (et vos fichiers nécessaires) dans un seul dossier.
Le compresser et l'enregistrer sous la forme prenom_nom_siteTuring.zip et l'envoyer par courriel.

  • Le site W3Schools pourra vous être utile.
  • Pensez à gérer votre mise en forme : les css seront présents dans les entêtes ou, encore mieux, dans un fichier .css.
  • N'oubliez pas d'adapter les balises META dans l'entête des fichiers.
  • Vous pouvez vérifiez la correction de votre fichier html à cette adresse.
  • Le codage est en quelque sorte la manière dont votre éditeur de texte a enregistré votre page web. Il a une influence notamment sur les caractères autorisés dans votre document.
    Selon l'éditeur de texte utilisé, le codage est indiqué dans la fenêtre de dialogue qui apparaît lorsqu'on fait 'Enregistrer Sous'. Si le choix vous est offert, préférez un encodage Unicode (UTF-8).
