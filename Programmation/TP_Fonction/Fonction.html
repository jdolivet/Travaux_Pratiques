Fonction

On souhaite construire une fonction escalier() prenant comme paramètre un entier naturel n.


Cette fonction escalier(n) doit:

  • imprimer "TypeError" si n n'est pas de type int. Dans ce cas, elle devra retourner -1.
  • imprimer "ValueError" si n est un entier strictement négatif. Dans ce cas, elle devra retourner 0.
  • imprimer un escalier possédant n marches avec un espace avant la première marche. Dans ce cas, elle devra retourner 1.
